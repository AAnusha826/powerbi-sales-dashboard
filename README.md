# powerbi-sales-dashboard
Power BI dashboard analyzing sales performance

Overview

This project is an interactive Power BI dashboard designed to analyze sales performance, profitability, and customer trends. It helps identify key insights to support business decisions.

Key Features

- KPI metrics: Total Sales, Total Cost, Gross Profit, Gross Margin
- Monthly sales trend analysis
- Product-wise sales breakdown
- Salesperson performance comparison
- Customer count tracking

Tools & Technologies

- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling

Data Model

- Designed a structured data model with relationships between tables
- Implemented calculated measures such as Total Sales, Profit, and Margin %
- Followed best practices for efficient reporting

Insights

- Identified top-performing products and salespersons
- Analyzed monthly sales fluctuations
- Evaluated profit margins across different categories

File

- "AnushaAP3_Simulated_Consulting_Project.pbix"

How to Use

1. Download the ".pbix" file
2. Open using Power BI Desktop
3. Explore the dashboard and interact with visuals

This project demonstrates skills in data analysis, dashboard design, and business intelligence using Power BI.
